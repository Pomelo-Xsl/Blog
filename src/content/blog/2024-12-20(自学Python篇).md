内置全局变量： __name__
语法：__name__ = "__main__" （即自己执行自己）
作用：判断当前模块是否为主模块，如果是，则执行该模块中的代码，否则不执行。

包：是项目结构中的文件夹或者目录,本质就是一个模块,也可以包含其他模块
与普通文件夹不同，包中必须包含一个__init__.py文件，该文件可以为空，但不能省略。
作用：可以将相关的模块、类、函数等组织在一起，便于管理和使用。（有效避免命名冲突,让结构更加清晰）
导包时,首先执行包内的__init__.py文件，然后再导入模块。

递归：在函数中调用自身，称为递归。
缺点：耗内存,运行效率低
普通函数
def add():
    s = 0
    for i in range(1,101):
        s += i
    return s

递归函数
def factorial(n):
    if n == 1:     # 递归出口
        return 1
    else:
        return n * factorial(n-1)   # 递归调用,递归规模比上一次减少

斐波那契数列
def fib(n):
    if n <= 1:
        return n
    else:
        return fib(n-2) + fib(n-1)
闭包
构成的条件
1.函数嵌套（函数里面再定义函数）
2.内层函数使用外层函数的局部变量
3.外层函数的返回值是内层函数的函数名 
简单的闭包函数：
def outer() :     #外层函数
    n = 10        #外层函数的局部变量 
    def inner():    #内层函数
        print(n)    #内层函数使用外层函数的局部变量
    return inner    #返回值是内层函数的函数名

print(outer())   #返回的是内部函数的内存地址
outer()()        #调用内部函数
或者可以这样写：
ot = outer()
ot()

带参数的闭包函数：
def outer(x) :     #外层函数
    y = 10        #外层函数的局部变量 
    def inner():    #内层函数
        print(x+y)    #内层函数使用外层函数的局部变量
    return inner    #返回值是内层函数的函数名,注意是名字,不是inner(),因为参数多或受限制时写法不规范

调用：
ot = outer(5) #调用函数
ot(2)           #调用内函数

id()函数：
判断两个变量是否是同一个值的引用（内存地址）

a = 10  #a里面存了10这个数值的引用
print(id(a))   #输出10这个值的内存地址
a = 20  #a里面存了20这个数值的引用
print(id(a))   #输出20这个值的内存地址

所以,两个id(a)不一致,值改变了,内存地址也改变了。

装饰器：本质是一个闭包函数,在不改变原函数的情况下,对其功能进行增强。（升级功能）
语法：@装饰器名  
条件：
1.不修改原程序或函数代码
2.不改变原函数的调用方法
例子：
def test(fn):  #fn是普通形参
    print("登录")
    print("注册")
    fn()   #  调用test2()
def test2():  # 增加的新功能
    print("发消息")
test(test2)   # 调用test()

标准版装饰器：
# 被装饰函数
def send():
    print("发消息")
# 闭包函数
def outer(fn):  # 外层函数
    def inner():  # 内层函数
        print("登录")
        fn()   # 调用被装饰函数
    return inner  # 返回内层函数的引用
ot = outer(send)  # 调用装饰器
ot()   # 调用内层函数

装饰器的原理就是将原有的函数名重新定义为以原函数为参数的闭包

语法糖：
因为这样写比较麻烦,所以python提供了语法糖@装饰器名,简化装饰器的写法。

def outer(fn):  # 外层函数
    def inner():  # 内层函数
        print("登录")
        fn()   # 调用被装饰器
    return inner  # 返回内层函数的引用

@outer  # 装饰器,注意不要加(),前者是引用,后者是调用,会返回原函数要返回的值
def send():
    print("发消息")

send()   # 调用装饰器

当 被装饰的参数有参数(一致原则)
def outer(fn):  # 外层函数
    def inner(name):  # 内层函数
        print(f"{name}是inner函数中的参数")
        fn()   # 调用被装饰器
    return inner  # 返回内层函数的引用
@outer  # 装饰器
def func(name):
    print("这是被装饰的函数")
func("张三")

多个装饰器
@outer
@inner
def func(name):
    print("这是被装饰的函数")

说明：由内到外先执行,即先执行@inner,再执行@outer,最后执行func()
